사실 fullpage 스크롤 내 기준에선 매우 귀찮고 어려운 것 같다.
근데 찾아보다 보니 쉬운 방법들도 있는 것 같다.
이 코드는 오직 HTML과 CSS만으로 작성되었다.

일단 HTML의 구조는 다음과 같다.

```html
<div id="fullpage">
	<div class="section">
		<1 />
	</div>
	<div class="section">
		<2 />
	</div>
	<div class="section">
		<3 />
		<Footer />
	</div>
</div>
```

CSS는 다음과 같다

```css
#fullpage {
    scroll-snap-type: y mandatory;
    overflow-y: scroll;
    height: 100vh;
    scroll-behavior: smooth;
}
.section {
    scroll-snap-align: start;
    min-height: 100vh;
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
}
```
